# 📱 Google Play Store Exploratory Data Analysis (EDA)

# 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Google Play Store Apps Dataset** to understand app performance, user ratings, reviews, categories, pricing models, and market trends.

The analysis focuses on data cleaning, statistical exploration, aggregation, and extracting business insights using Python and Pandas.

---

# 🎯 Objectives

- Explore the Google Play Store dataset
- Perform data cleaning and preprocessing
- Analyze app ratings and reviews
- Compare Free vs Paid applications
- Identify top-performing app categories
- Find the most reviewed applications
- Answer business-oriented analytical questions
- Generate actionable insights using data visualization

# 🔍 Key Insights

- ⭐ Average App Rating: **4.19 / 5**
- 📱 Total Apps: **10,841**
- 📂 Categories: **34**
- 💰 Free Apps: **92.6%**
- 💲 Paid Apps: **7.4%**
- 🏆 Highest Rated Category: **Events**
- 📚 Education apps also receive excellent ratings.
- 📢 Facebook has the highest number of reviews.
- ⭐ 274 applications have a perfect 5.0 rating.
- 🔮 Only 3 apps contain **"Astrology"** in their title.
